# 🎓 School Result Portal

A web-based primary school result portal that allows students to check their results using access codes after payment confirmation.

---

## 🚀 Features

- Code-based result access system
- Printable A4 report sheet
- Mobile-friendly design
- Clean and simple user interface

---

## 📂 Project Structure

index.html – Homepage  
report-card-payment.html – Payment & code entry page  
result.html – Result display page  
/style/ – CSS files  
/data/ – Images and videos
